Introduction

JQuery's Urdu Editor plugin provides the capability of converting any edit area on a web page into an Urdu edit area. It makes it possible to type Urdu inside the edit areas without the need of installing Urdu language support or the Urdu keyboard in the operating system. This work is derived from Urdu WebPad which is a javascript component that also provides the capability of typing Urdu in a text area on the web page without the need of installing Urdu language support or the Urdu keyboard in the operating system. Creating an Urdu Editor plugin for jQuery is a significant development since it makes it possible to use all the powerfull features of the jQuery framework alongwith the Urdu editing capabilities.

Including the Necessary Script References

First of all you need to download the latest version of the jQuery libary which you can get here. You also need to downlaod the zip-file containing the Urdu Editor plugin for jQuery. Then you can include the references to the jQuery library and the Urdu Editor plugin in the following way:

<html>
<head>
    <link rel="stylesheet" type="text/css" href="UrduEditor.css" />
    <script src="jquery.js" type="text/javascript"></script>  
    <script src="jquery.UrduEditor.js" type="text/javascript"></script>  
</head>
<body>
<!-- Text areas will added later -->
</body>                                                                 
</html>

Converting edit areas into Urdu edit areas

Selecting a text area for conversion into an Urdu editor is done by the means of jQuery selectors. The fastest way to select a html-element with jQuery is by using the id-attribute of the element. Most of the jQuery operations can be performed inside a ready event that is registered in the following way:

$(document).ready(function() {
   // do stuff when DOM is ready
 });

The ready event then signals the completion of the html-content. Therefore any html-manipulation can be done inside this event handler. Consider for example the following html which contains three edit areas:

The above html contain edit areas each with the id attribute defined. These edit areas can be converted into Urdu edit areas by putting the following code inside the head tag:

$(document).ready(function() {  
	$("#username").UrduEditor("14px"); 
	$("#subject").UrduEditor("14px"); 
	$("#message").UrduEditor("18px");               
});

Alternatively, following code can be used to achieve the same effect: 

$(document).ready(function() {  
	$('input').UrduEditor("14px"); 
	$('textarea').UrduEditor("18px"); 
});

Note how jQuery selectors are used in different ways to manipulate the edit areas. The following figure shows the sample web page after converting the edit areas into Urdu editor:

Controls for switching language mode

The Urdu editor plugin for jQuery provides the CTRL+SPACE keyboard shortcut for switching the language mode from Urdu to English and vice versa. It also provides the writeToggleControl for inserting radio button controls for switching the language mode for a particular edit area. The following code snippet illustrates the use of the writeToggleControl function: 

<tr>
    <td class="table"><input id="username" name="username" type="text" size="30">
		<br /><script type="text/javascript">
			$(this).UrduEditor.writeToggleControl('username');
		</script>
	</td>
  </tr>
  <tr>

	</td>

  <tr>

Using the On-Screen Keyboard

The Urdu editor plugin for jQuery provides the writeKeyboard for putting an on-screen keypad to assist in typing Urdu. The following code snippet illustrates the use of the writeKeyboard function: 

<script type="text/javascript">
	$(this).UrduEditor.writeKeyboard();
</script>

Using the Virtual Keyboard

The Urdu editor plugin for jQuery also includes a tailored version of the virtual keyboard found here. In order to use the virtual keyboard, its javascript and stylesheet references must be included in the web page as shown in the following: 

<link rel="stylesheet" type="text/css" href="keyboard.css" />
<script src="keyboard.js" type="text/javascript"></script>

After that the virtual keyboard can be attached to any edit area by adding "keyboardInput" to its class attribute. This can be achieved by using the addClass function of jQuery as shown in the following code: 


<script language="JavaScript" type="text/javascript">
	 $(document).ready(function() {  
		$('#username').UrduEditor("14px"); $("#username").addClass("keyboardInput");
		$('#subject').UrduEditor("14px");$("#subject").addClass("keyboardInput");
		$('#message').UrduEditor("18px"); $("#message").addClass("keyboardInput");
      });
</script>


Customizing Urdu Editor Plugin's Defaults

The Urdu Editor plugin for JQuery also lets you customize certain attributes of the Urdu text areas. By default any Urdu edit area uses the Urdu Naskh Asiatype font, the background color in Urdu mode is #CCCCFF and the background color in English mode is #99FF99. Following is an example of customizing the editor defaults:

<script type="text/javascript"> 
   $.fn.UrduEditor.defaults.EditorFont = 'Alvi Nastaleeq';
   $.fn.UrduEditor.defaults.UrduColor = "#CCCC66";
   $.fn.UrduEditor.defaults.EnglishColor = "#FFCCCC";
</script>

Resizable textarea

The resizable textarea plugin of jQuery can also be used in conjunction with the Urdu editor plugin. In order to make a textarea resizable include the script reference of this plugin as shown in the following:

 

	<script type="text/javascript" src="jquery.textarearesizer.js"></script>

Further, the following style definitions should be included in the stylesheet of the web page:

 

	div.grippie {
		background:#EEEEEE url(grippie.png) no-repeat scroll center 2px;
		border-color:#DDDDDD;
		border-style:solid;
		border-width:0pt 1px 1px;
		cursor:s-resize;
		height:9px;
		overflow:hidden;
	}
	.resizable-textarea textarea {
		display:block;
		margin-bottom:0pt;
		width:95%;
		height: 20%;
	}

Now any textarea can be made resizable by using the TextAreaResizer() function as shown in the following:

 

$(document).ready(function() { 
	$("#message").TextAreaResizer();
});
